📈 Dashboard Visualizations

The dashboard includes the following charts:

🎬 Genre Distribution

Shows the most common genres available on Netflix.

⭐ Rating Distribution

Displays the distribution of titles by audience rating categories.

📈 Content Growth Over Time

Shows how the number of Netflix titles has increased over the years.

🌍 Country Contribution

Displays which countries produce the most Netflix content.

These charts are interconnected through filters, allowing users to analyze the dataset dynamically.

🛠️ Technologies Used
Python
Pandas – Data cleaning and manipulation
NumPy – Numerical computations
Matplotlib & Seaborn – Data visualization
Plotly – Interactive charts and dashboard
WordCloud – Text visualization
Jupyter Notebook – Development environment
